# 血染钟楼说书人助手 AI (BotC Storyteller AI Assistant)

这是一个轻量级的网页版小程序，旨在为《血染钟楼》(Blood on the Clocktower) 的说书人提供实时的 AI 控场和局面平衡建议。

## 功能介绍
- **局势诊断**：根据当前游戏天数（前期/中期/决战）和场面优劣势（好人优势/邪恶优势），快速判断当前游戏状态。
- **动态建议生成**：结合剧本中的特殊可控变量（如醉酒、中毒、疯狂、复活、外乡人机制），针对性地给出如何发放信息、如何裁决边缘规则的平衡策略。
- **零配置，秒响应**：纯静态前端应用，结合 Cloudflare Workers 部署，提供极速响应。

## 部署与使用
本项目可以作为静态单页应用 (SPA) 运行，直接打开 `index.html` 即可使用。

同时完美支持一件部署至 **Cloudflare Workers** 或 **Vercel** 边缘节点，适用于移动端浏览器在带局时快速使用。
